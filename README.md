# NLP_csv_analyzer
이 repository는 NLP 대회 뿐 아니라, NLP task와 관련된 모델의 학습/추론 결과를 분석하기 위한 라이브러리입니다.

## available Task
- regression (STS) [진행중]
- multi class classification (Relation Extraction) [진행중]
